Reference Agent Playground

Simulate a third-party agent integrating with Mems.

This playground uses the public APIs the same way an external agent would: read session context, search long-term memory, build a prompt, answer, then write the user and assistant turns back into memory.

Flow

`GET /memories/context` -> `POST /memories/search` -> LLM/fallback -> `POST /memories/turns`

Debug Surface

Prompt, context window, search hits, write result, and raw simulator response are visible on the same page.
